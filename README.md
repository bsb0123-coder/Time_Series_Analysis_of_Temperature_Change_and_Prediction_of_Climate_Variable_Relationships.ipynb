# ● 시계열 기온 변화 분석 및 기후 변수 상관관계 예측 
**기간: 2000.01 ~ 2025.10**  
**지역: 서울, 강릉, 대구, 창원, 제주**  
**분석 변수: 평균기온, 평균상대습도, 월합강수량, 일사량**

---

## ● 프로젝트 개요
이 프로젝트는 기상청에서 제공하는 월 단위 기상 데이터를 기반으로  
한국 5개 지역(강릉, 서울, 대구, 창원, 제주)의 기온·습도·강수량·일사량을 분석하고,  
각 기후 변수들이 어떻게 변화해왔는지를 시각적으로 탐색하는 것을 목표로 한다.
나아가 기후 변수들의 상관관계 분석 및 회귀 모델을 구축해보고 예측값을 구해본다.

---

## ● 데이터 출처
- 출처: 기상청 기후자료개방포털  
- 데이터 종류: **월단위 평균기온, 평균상대습도, 월합강수량, 일사량**
- 데이터 기간: 2000.01 ~ 2025.10  
- 파일명: '2000.01~2025.10 월단위 평균온도, 습도, 강수량, 일사량 자료 (기상청).csv' → '전처리 후: 전처리 이후.csv'

---

## ● 전처리 과정 요약
1. 컬럼명 한글 → 영어로 변환  
2. 'Date_raw'를 'YYYY-MM' 형태의 월 단위 날짜로 변환  
3. 분석 지역 5곳만 필터링  
4. **창원(2000.01~2015.08)의 일사량 결측치를 같은 기간의 다른 4개 지역 평균값으로 보정**
5. 남은 결측치는 station+month 평균값으로 추가 보정  
6. 최종 전처리 결과 저장 → '전처리 이후.csv'

---

## ● 분석 내용

### 1) 지역별 시계열 분석  
- 기온, 습도, 강수량, 일사량 변화 추세 시각화  
- 각 지역의 계절성·기후 특성 파악  

### 2) 연도별 추세 분석
- 연평균 기온/습도/강수/일사량 변화를 연도 기준으로 확인  
- 장기적 기후 변화 경향 해석

### 3) 상관관계 분석 (Heatmap)
- 4개 기후 변수(temperature, humidity, precipitation, sunshine)의 상관관계를 히트맵으로 표현  
- 기온과 어떤 변수가 상관관계를 어떻게 갖는지 파

### 4) 회귀 분석 (기온 → 습도)
- 각 지역의 ‘기온 상승 시 습도 변화’와 '기온 상승 시 일사량 변화' 관계를 선형회귀로 분석  
- 기온의 변화에 따른 두 기후 변수의 변화 예측
